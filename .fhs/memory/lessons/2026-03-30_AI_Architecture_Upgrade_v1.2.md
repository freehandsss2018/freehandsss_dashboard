## 避坑指南
- **路徑陷阱**：在 `scripts/` 下執行 Node 腳本時，必須注意 `__dirname` 相對於專案根目錄的路徑偏移。
- **依賴失蹤**：環境中若缺少 `dotenv`，腳本會崩潰。已手動補齊。
- **雙源衝突**：任何規則如果同時存在於兩個地方（如舊 Prompt 內含邏輯，新指令也含邏輯），必須使用 Router 格式導向單一來源。
