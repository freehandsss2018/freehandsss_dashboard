# Lesson: IG 預覽文字格式細節調整 (2026-04-18)

## 背景
用戶要求對 IG 訊息預覽進行多維度的格式微調，涉及標題、符號、括號與空格。

## 關鍵教訓
1. **單號格式要求精確**: 用戶希望單號與產品名稱之間有空格，且 `#` 後方也需要空格，形如 `(訂單編號# 0631044 吊飾產品)`。
2. **符號與括號風格**: 用戶明確要求將全角括號 `（ ）` 換成半角括號 `( )`。
3. **Emoji vs 文字**: 在正式預覽中，用戶傾向移除過多的裝飾性 Emoji（如 `✨`, `🖼️`, `⭐️`），改用簡潔的橫槓 `-` 作為條款列示符。
4. **術語統一**: 將「金屬產品」統稱為「吊飾產品」以符合最新的品牌定位。

## 實作注意
- 對 V37 (Stable) 與 V39 (Dev) 的同步修改是維持一致性的關鍵。
- 標題列（noticeA/B 抬頭）不應帶有橫槓 `-`，僅內容行需要。
