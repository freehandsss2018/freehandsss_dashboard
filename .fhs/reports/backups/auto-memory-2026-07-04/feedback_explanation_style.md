用實物 + 數字表格解說，不用抽象術語。

**Why:** Fat Mo 表示「bundle」、「SKU mapping」等系統術語難以理解，直接看「客人買了什麼 + 成本怎樣算」更清晰。

**How to apply:** 解釋成本/定價時，格式如下：

| 部位 | 件數 | 成本 | 說明 |
|------|------|------|------|
| 左手 | 2 | $290 | 買 2 個同部位，用 2飾批次價 |
| 右腳 | 2 | $290 | 同上 |
| 跨部位扣減 | | −$20 | 同一客人同類產品 |
| **最終成本** | | **$560** | |

避免用：bundle、SKU mapping、Order_Items qty semantics 等術語。
